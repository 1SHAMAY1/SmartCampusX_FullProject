
# SmartCampusX

SmartCampusX is a modular platform for managing campus activities like attendance, room booking, and safety alerts.

## Technologies
- Frontend: React + TailwindCSS
- Backend: Node.js + Express + MongoDB
- Deployment: Vercel + Render

## Setup

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Deployment
- Backend: Render.com
- Frontend: Vercel.com
